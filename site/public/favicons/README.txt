Favicons for Carinya Parc Website
===============================

The following favicon files need to be generated and placed in this directory:

- favicon.ico (multi-size ICO file, place in the public root)
- favicon-16x16.png (16x16px)
- favicon-32x32.png (32x32px)
- favicon-192x192.png (192x192px)
- favicon-512x512.png (512x512px)
- apple-touch-icon.png (180x180px)

You can generate these files using your logo and a favicon generator like:
- https://realfavicongenerator.net/
- https://favicon.io/

Instructions:
1. Upload your logo (preferably a square, simple version of your logo)
2. Configure the settings for each platform
3. Download the generated package
4. Place the files as named above in this directory and the public root

These favicons will enable:
- Standard browser tab icons
- Bookmark icons
- Mobile home screen icons
- PWA (Progressive Web App) icons

Note: The manifest file (site.webmanifest) references some of these icons.
